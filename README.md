# 🌸 Flower Bouquet E-commerce Website

A simple flower bouquet selling website built using PHP and MySQL.
This project is created for learning and portfolio purposes.

## ✨ Features
- Product listing (Rose, Lily, Tulip)
- Custom bouquet creation
- Shopping cart system
- Admin panel (CRUD)
- Responsive design (Bootstrap)

## 🛠 Technologies
- HTML, CSS, Bootstrap
- PHP
- MySQL

## ⚙️ How to Run
1. Clone the repo
2. Import database to MySQL
3. Configure `db.php`
4. Run on localhost (XAMPP)
